colour selection

Colour selection for use on category page quick add, and filtering.

The input field of type checkbox and radio can be used. The input must have the class .compact-cs-input. There must be a label sibling with the class .compact-cs-label containing an inline span with the class .compact-cs-circle. Use checked to choose the initial option.

The colour name may be added to a div next to .compact-cs-circle with a class name of .compact-cs-name.
